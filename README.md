# Projeto-TCC-UFLA

Esse repositório foi criado para documentar e preservar o código utilizado para o desenvolvimento do meu TCC (Trabalho de Conclusão de Curso) na Universidade Federal de Lavras, cujo objetivo foi de avaliar textos alternativos gerados por diversos modelos de IA no contexto de imagens encontradas em noticiários a fim de entender se são úteis para o usuário. Esse é um trabalho que mistura Inteligência Artificial com a àrea de Interação Humano-Computador, focado especificamente na Acessibilidade Digital.
